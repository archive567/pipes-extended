Some pipes helpers and rational structure for the pipes zoo.

## Develop

``` sh
$ git clone https://github.com/tonyday567/pipes-extended.git
$ cd pipes-extended

$ cabal sandbox init
$ cabal configure
$ cabal install --dependencies-only --dry-run
$ cabal install --dependencies-only
$ cabal build
```
